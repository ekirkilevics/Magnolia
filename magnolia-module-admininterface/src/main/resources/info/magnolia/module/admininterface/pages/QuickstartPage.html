Now install some templates

Templates are skeletal web pages (or sections of web pages) used to display content. Since you typically will want to publish content on the web - Magnolia is actually used in many different ways -, you should now install or define some page and paragraph templates.

We highly recommend to use our best practice, state of the art templates provided by the Supersonic templating bundle. These have quickly become the standard way to build highly customized, standards compliant and accessible web sites using Magnolia that look good and work perfectly across multiple platforms and devices. Alternatively, you can start building your own templates now.
