Nightly builds

Nightly builds on this page have been automatically generated and are not guaranteed to be stable. It is possible that these builds have major bugs and may not run at all. Be sure to check junit test results to understand what is working and what is not in these drops. (Hint: if the repository bootstrap test failed, you will probably not be able to run the war).

Nightly build always include the current jackrabbit version, compiled from svn. The version of the jackrabbit jar in magnolia repository can be different.
